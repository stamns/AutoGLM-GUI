# 从零搭建一个工作流

本教程将带你从零创建一个多步骤工作流，完成采集→分析→控制的自动化闭环。

## 前置条件

- 已完成设备连接（二维码/USB/WiFi）
- 已配置必要的模型或服务（如有）

## 步骤一：创建工作流

1. 在工作流页面点击「新建」
2. 设定名称与目标，选择基础模板或从空白开始

## 步骤二：添加采集节点

1. 添加「采集」步骤，选择数据源（摄像/屏幕/传感）
2. 配置采样率与画面区域（如适用）

## 步骤三：添加分析节点

1. 添加「分析」步骤，绑定模型或服务
2. 配置阈值、上下文与超时策略

## 步骤四：添加控制节点

1. 添加「控制」步骤，设定设备动作（点击、拖拽、手势）
2. 配置失败重试与安全确认

## 调试与验证

- 使用模拟运行检查各节点参数与输出
- 调整顺序与分支，确保满足业务逻辑

## 执行与监控

- 执行工作流并在历史记录中查看状态、日志与结果
- 对失败节点进行重试或回滚

## 扩展建议

- 将稳定工作流保存为模板供团队复用
- 为关键步骤开启审计与权限校验
