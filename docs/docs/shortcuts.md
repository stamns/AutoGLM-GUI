# 快捷键与交互手势

本页列出常用快捷键、页面级操作与手势，帮助你更高效地使用 AutoGLM-GUI。

## 全局快捷键

- `Ctrl/Cmd + K`：打开命令面板
- `Ctrl/Cmd + /`：打开快捷说明或帮助
- `Ctrl/Cmd + S`：保存当前配置或工作流

## 页面级操作

- 对话与任务：快速新建任务、切换对话模式、查看历史记录
- 工作流编辑：添加节点、调整顺序、启用/禁用节点

## 交互手势

- 点击/拖拽：在实时画面中进行精准操作
- 滑动/长按：执行页面或设备上的常见手势

## 可定制项

- 在设置中自定义部分快捷键与手势敏感度
- 导入导出快捷配置以便团队统一
