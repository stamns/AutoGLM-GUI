# 连接设备

详细介绍如何将手机等设备与 AutoGLM-GUI 建立可靠连接。

## 二维码无线配对

- 在应用中打开配对页，展示二维码
- 使用手机端扫描二维码，自动完成发现与绑定
- 常见问题：
  - 不同网络：确保手机与电脑在同一局域网或支持穿透策略
  - 防火墙：放行应用端口（见 FAQ）

## USB 连接

- 通过数据线连接手机与电脑，授权调试/数据传输
- 安装必要驱动并确认系统识别设备
- 优点：稳定性高、延迟低；缺点：需要物理连接

## WiFi 连接

- 在同一网络下，应用自动发现设备或手动输入地址
- 若路由器隔离子网或禁止广播，建议改用二维码或 USB

## 连接状态与重连

- 在设备列表中查看在线状态与信号质量
- 断线后自动重试；如失败，检查网络与权限设置
