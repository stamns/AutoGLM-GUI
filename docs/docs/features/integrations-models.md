# 模型与服务集成

介绍可用的模型/外部服务以及如何在 AutoGLM-GUI 中进行配置与使用。

## 支持范围

- 模型：本地/云端模型服务（按需配置密钥与地址）
- 外部服务：存储、分析、推理等通用能力

## 配置方式

- 环境变量：`API_KEY`、服务地址与配额设置
- 应用设置：在界面中选择模型与服务并保存配置

## 使用示例

- 在工作流节点中选择「分析」或「推理」步骤，绑定目标模型
- 在对话模式下调用指定服务生成结果或控制指令

## 常见错误与排查

- 身份校验失败：检查密钥是否正确与配额是否充足
- 超时/网络问题：确认服务地址可达并优化重试策略

## 最佳实践

- 根据场景选择合适的模型与资源预算
- 为不同环境（开发/生产）维护独立的密钥与配置
